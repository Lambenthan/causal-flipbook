# 第 9 章 · 异质性效应：因果森林与个体化 CATE

**本章目标**

- 看清 ATE 把 5735 名患者的个体效应压缩成一个数字带来的信息损失
- 理解因果森林的核心机制：诚实分裂与双重稳健评分
- 在 RHC 数据上用 grf 估出每名患者的 CATE，看 RHC 在不同人群里效应方向与幅度如何分布
- 用变量重要性和分位数分组识别哪些临床特征预测异质性
- 区分"探索性异质性发现"与"政策可执行的亚组结论"，避免亚组分析的常见陷阱

---

第 8 章用 E-value 把 RHC 的平均效应稳健性压力测了一遍。结论是要让 RHC 增加 180 天死亡率这条结论翻盘，未测量混杂需要达到 3 倍 APACHE 强度，临床上不易出现。这一章把镜头从"RHC 平均增加 4.5 个百分点死亡率"放回到具体的 5735 名患者上。ICU 收治的患者差异极大，从 20 岁的多发外伤患者到 90 岁的多器官衰竭患者，从 APACHE 评分 20 的轻症到 APACHE 评分 80 的重症，把他们的个体效应取均值得到 0.045 这个数字，过程里抹掉了所有差异。一个 ATE = 0.045 的世界里，可能是每个人都被 RHC 害了 4.5 个百分点，也可能是 80% 的患者被害 6 个百分点、20% 的患者获益 1.5 个百分点。两种情况下 ATE 完全相同，但临床意义截然不同——后者意味着应该挑出获益的 20%，让他们继续插 RHC，禁止给其他人插。

回答"RHC 对哪些人有害、对哪些人可能有益"的问题，需要从平均处理效应 ATE 切到条件平均处理效应 CATE，即 $\tau(L) = E[Y(1) - Y(0) \mid L]$ ，每个协变量层都有自己的处理效应。Athey 与 Wager 2018–2019 年提出的因果森林是当前估计 CATE 的主流工具之一。本章先讲清楚 ATE 与 CATE 的差距，再讲因果森林的核心机制，最后在 RHC 数据上跑出 5735 个个体化 CATE，看异质性的临床信号长什么样。

## 9.1 从平均到个体：ATE 的信息损失

ATE 把全部个体效应压缩成单一标量，这种压缩在两种情况下会误导决策。

第一种是处理效应有方向相反的子群。如果 RHC 对老年重症患者明显有害（CATE = +0.10）、对年轻轻症患者明显有益（CATE = -0.05），整体 ATE 可能是 +0.045，看起来"略微有害"，但这个数字既不代表老年人也不代表年轻人。临床医生看到 ATE = +0.045 后决定"全部停掉 RHC"，会让那些可能获益的年轻轻症患者被过度保守地剥夺一个实际有用的工具。

第二种是处理效应在不同子群幅度差异极大。哪怕 CATE 在所有子群都是正的，幅度从 +0.02 到 +0.10 不等，临床决策仍然应当区别对待——CATE = +0.02 的子群可能值得权衡 RHC 的诊断信息收益，CATE = +0.10 的子群明显得不偿失。ATE 给出的"平均 +0.045"覆盖不了这种差异。

> [!IMPORTANT]
> **命题 9.1 — ATE 与 CATE 的关系**
>
> 在协变量分布 $P_L$ 下， $\mathrm{ATE} = E_L[\tau(L)] = \int \tau(l) dP_L(l)$ 。ATE 是 CATE 的全人群边际化。两者关系如下：
>
> - ATE 报告的是全样本的政策效应（如果整体接受 RHC vs 整体不接受）
> - CATE 报告的是个体的政策效应（这位 78 岁、APACHE 62 的患者接受 RHC 比不接受死亡率高多少）
>
> 当 CATE 在所有子群都同号且幅度接近时，ATE 是足够代表性的指标；当 CATE 方向相反或幅度差异显著时，ATE 单独使用会导致政策错配。

精准医疗这个概念在过去十年的核心驱动力就是把决策从"基于 ATE"切换到"基于 CATE"。华法林对某些患者降低卒中风险 65%，对另一些引发严重出血；抗肿瘤免疫治疗对 PD-L1 高表达者可能完全缓解，对低表达者可能引发免疫性肺炎。这些场景下 ATE 这个数字几乎没有政策含义，研究者真正关心的是"哪些个体应该获得这个处理"。

## 9.2 因果森林的核心机制

<div align="center">
  <img src="../assets/chap09/02_honest_split.gif" alt="诚实分裂：决定树结构与估计叶子效应分到不同子样本" width="720" />
</div>

随机森林估计回归函数 $E[Y \mid L]$ 在预测任务里很成熟，但直接搬到因果推断会遇到两个困难。第一，随机森林训练时用 $Y$ 作为分裂标准，不会按"处理效应大小"决定分裂位置。第二，随机森林的叶子均值在训练集上是过拟合的，把它直接当 $\hat\tau(L)$ 用 CI 失效。Athey、Imbens 与 Wager 在 2018 年的 *Annals of Statistics* 论文里给出两条修正，把随机森林改造成因果森林。

第一条修正叫**诚实分裂**。把数据随机分成两份， $\mathcal S_1$ 用来决定每棵树的分裂位置（即决定树的拓扑结构）， $\mathcal S_2$ 用来在每个叶子上估计该叶子的 $\hat\tau$ 。同一份数据不同时用于结构选择和效应估计，避免训练集上的偏差被搬运到效应估计里。这条性质让因果森林的渐近正态性可以严格证明，CI 有公式可算。

第二条修正叫**双重稳健评分**。每个观察的"伪结局"换成 AIPW 风格的影响函数得分
$$\Gamma_i = \hat\tau(L_i) + \frac{A_i - \hat e(L_i)}{\hat e(L_i)(1 - \hat e(L_i))}\big(Y_i - \hat m(A_i, L_i)\big),$$
其中 $\hat\tau(L_i)$ 是初始 CATE 估计， $\hat e(L_i)$ 是倾向得分， $\hat m$ 是结果模型。 $\Gamma_i$ 在 $\hat e$ 与 $\hat m$ 任一正确时都是 $\tau(L_i)$ 的无偏估计，与 AIPW 共享双重稳健性。把 $\Gamma_i$ 喂进随机森林分裂规则，森林学到的就是"在哪些协变量上切开，能让两个子叶子的处理效应差异最大"。

> [!NOTE]
> **定义 9.1 — 因果森林（Athey-Wager 2019）**
>
> 因果森林是随机森林的因果推断版本，每棵树满足两个条件：
>
> **诚实分裂**：训练数据随机分为决定树结构的子样本 $\mathcal S_1$ 与估计叶子效应的子样本 $\mathcal S_2$ ，两个子样本不重叠。
>
> **基于影响函数的分裂**：分裂规则最大化两个子叶子之间 AIPW 影响函数得分 $\Gamma_i$ 的差异，等价于在协变量空间里寻找让 CATE 异质性最大化的分割。
>
> 个体 CATE 估计为
>
> $$\hat\tau(l) = \frac{\sum_{i \in \mathcal{N}(l)} w_i(l) \cdot \Gamma_i}{\sum_{i \in \mathcal{N}(l)} w_i(l)},$$
>
> 其中 $\mathcal{N}(l)$ 是落在与 $l$ 同叶子的训练样本， $w_i(l)$ 是森林平均权重。

因果森林还有一条工程上重要的性质，**自适应核函数**。每棵树把协变量空间切成多个矩形叶子，叶子边界由数据自动决定。在协变量空间的不同区域，森林的"邻域大小"自动调整：协变量密集的区域叶子小，协变量稀疏的区域叶子大。这让因果森林同时具备非参数估计的灵活性和局部加权回归的稳定性。

## 9.3 在 RHC 上估 CATE

<div align="center">
  <img src="../assets/chap09/03_cate_histogram.gif" alt="RHC：5735 个 CATE 的分布形状" width="720" />
</div>

```r
set.seed(2026)
library(tidyverse); library(grf)

d <- read_csv(here::here("data", "rhc.csv"), show_col_types = FALSE) |>
  mutate(death180_bin = if_else(death180 == "Yes", 1L, 0L),
         rhc_bin      = if_else(rhc == 1, 1L, 0L),
         sex_bin      = if_else(sex == "Male", 1L, 0L),
         cancer       = as.integer(as.factor(cancer)) - 1L)

covs <- c("age", "sex_bin", "edu", "das_index", "apache_score",
          "glasgow_coma_score", "blood_pressure", "wbc", "heart_rate",
          "respiratory_rate", "temperature", "albumin", "hematocrit",
          "bilirubin", "creatinine", "weight", "cancer", "cardiovascular",
          "congestive_hf", "dementia", "psychiatric", "pulmonary",
          "renal", "hepatic", "gi_bleed", "tumor", "immunosupperssion",
          "transfer_hx", "mi")

X <- as.matrix(d[, covs]); A <- d$rhc_bin; Y <- d$death180_bin

# 训练因果森林（2000 棵树）
cf <- causal_forest(X = X, Y = Y, W = A, num.trees = 2000, seed = 2026)

# 全样本 ATE 与 CATE
cf_ate <- average_treatment_effect(cf, target.sample = "all")
tau_hat <- predict(cf)$predictions

# 分位数描述
quantile(tau_hat, c(0, 0.10, 0.25, 0.50, 0.75, 0.90, 1))
mean(tau_hat > 0); mean(tau_hat < 0)

# 变量重要性
vi <- variable_importance(cf)
tibble(var = covs, importance = as.numeric(vi)) |>
  arrange(desc(importance)) |> head(10)

# 按 APACHE 与年龄分位数分组
d$tau_hat <- tau_hat
d |>
  mutate(apache_grp = cut(apache_score,
                          breaks = c(-Inf, quantile(apache_score, c(0.25, 0.5, 0.75)), Inf),
                          labels = c("Q1 轻症", "Q2", "Q3", "Q4 重症"))) |>
  group_by(apache_grp) |>
  summarise(n = n(), cate_mean = mean(tau_hat), cate_sd = sd(tau_hat))
```

> [!TIP]
> **解读 — RHC 上 CATE 的整体分布**
>
> 因果森林给出的 ATE = 0.0442，SE = 0.0124，与第 6 章 AIPW 的 0.0455、第 7 章 DML 的 0.0445 高度一致，是横向稳健性的又一个确认。但 ATE 只是均值，CATE 的分布才是本章关心的对象。5735 个 CATE 的描述统计：均值 0.0425，标准差 0.0239，最小值 -0.0398，10% 分位数 0.0123，25% 分位数 0.0258，中位数 0.0421，75% 分位数 0.0590，90% 分位数 0.0736，最大值 0.1306。
>
> 关键诊断：96.65% 的患者 CATE 大于 0，3.35% 的患者 CATE 小于 0，17.28% 的患者 \|CATE\| 小于 0.02（接近零）。换句话说，"RHC 增加 180 天死亡风险"这个判断在 RHC 这份 ICU 患者群体上几乎是普适的——只有约 3.4% 的患者表现出"可能从 RHC 中获益"的信号，且最大获益幅度（CATE 最小值 -0.0398）也只有 4 个百分点。最大有害幅度可达 13 个百分点，意味着确实有一小群患者 RHC 让 180 天死亡概率上升 13 个点。CATE 分布偏右、长尾在正方向，这是 ATE = 0.045 这个均值背后的形状。

## 9.4 哪些变量驱动异质性

<div align="center">
  <img src="../assets/chap09/04_variable_importance.gif" alt="因果森林：变量重要性 Top 10" width="720" />
</div>

因果森林本身不会告诉研究者"为什么这位患者 CATE 高、那位低"，但它的变量重要性指标可以指出哪些协变量在分裂决策中被频繁使用。

> [!TIP]
> **解读 — RHC 上变量重要性 Top 10**
>
> 因果森林在分裂时最常用的前 10 个协变量是：bilirubin（胆红素）0.091、weight（体重）0.086、wbc（白细胞）0.082、hematocrit（红细胞压积）0.079、blood_pressure（血压）0.076、age（年龄）0.066、respiratory_rate（呼吸频率）0.064、temperature（体温）0.060、edu（教育）0.058、glasgow_coma_score（昏迷评分）0.055。胆红素、白细胞、红细胞压积、血压这些生理指标排在前列，说明 RHC 对死亡率的影响在不同肝功能、不同感染状态、不同贫血程度、不同血流动力学稳定性的患者身上有区别——临床上这与"RHC 在某些状态下能提供有用监测信息、在另一些状态下风险大于收益"的经验一致。

变量重要性是描述性的，不是因果的。它告诉研究者哪些变量在 CATE 估计中信息量大，但不告诉哪个变量"导致"了异质性。把"bilirubin 排名第一"解读成"高胆红素患者特别该禁用 RHC"是过度推断——重要性高也可能因为 bilirubin 与肝功能、感染、危重综合征等多个未观察机制相关，森林只是用它作为这些隐藏维度的代理。要进一步把异质性的临床机制讲清楚，需要把高 bilirubin 子群的其他特征也描述清楚，看是否落在某个临床综合征上。

## 9.5 临床基准的亚组验证

<div align="center">
  <img src="../assets/chap09/05_subgroup_cate.gif" alt="子群 CATE：APACHE 与年龄四分位数" width="720" />
</div>

变量重要性给出"森林觉得哪些变量重要"的信号，但临床读者通常想看到具体的子群对比，例如"老年患者 CATE 与年轻患者 CATE 差多少"。把 CATE 按 APACHE 评分四分位数与年龄四分位数分组取均值，可以给临床读者直接答案。

> [!TIP]
> **解读 — 按 APACHE 与年龄分组的 CATE**
>
> 按 APACHE 评分四分位数分组：Q1 轻症 (n=1512) CATE 均值 0.0385，Q2 (n=1471) 0.0404，Q3 (n=1321) 0.0463，Q4 重症 (n=1431) 0.0456。
>
> 按年龄四分位数分组：Q1 年轻 (n=1434) CATE 均值 0.0372，Q2 (n=1434) 0.0396，Q3 (n=1433) 0.0475，Q4 老年 (n=1434) 0.0460。
>
> 两条信号方向一致：APACHE 评分越高、年龄越大，RHC 的有害效应越强，从 0.038 上升到 0.046 这个量级。但差异并不悬殊——最高与最低组之间的 CATE 差距只有 0.008–0.009 个百分点，远小于 ATE 本身的量级 0.044。这说明 RHC 的有害效应在 ICU 患者整体上几乎是均匀分布的，没有出现某个明显获益的子群。
>
> Best Linear Projection（用 CATE 对几个主要协变量做线性回归）只检测到 albumin 系数 0.0354 显著（p = 0.020），其他协变量都不显著。这条结果支持 CATE 异质性确实存在但幅度有限的判断。

这条发现实际意义不小。它意味着研究者**不能**通过简单的临床特征筛出一个"应该插 RHC 的子群"。即便细分到 APACHE 第一分位数（最轻症）+ 年轻患者，CATE 仍然约 +0.037，没有翻到负的方向。从政策角度，"用临床特征做精准 RHC 决策"在当前数据支持下是不可行的——需要的是干脆停止常规 RHC 监测，或者去找到 RHC 数据库之外、目前没有测到的更细致的指标。

## 9.6 个体化 CATE 与床旁决策的距离

研究层面的 CATE 估计与临床床旁的"该不该给这位患者插 RHC"的决策之间还隔着相当远的距离。这条距离不是统计技术能完全填平的，但理解它的来源可以帮助研究者写出更负责任的论文表述、临床读者做出更冷静的解读。

距离的第一段来自统计精度。CATE 是 individual-level 的预测，每个 $\hat\tau(L_i)$ 的 95% CI 通常在 [-0.05, +0.15] 这个量级，远比点估计 0.04 宽。把 CATE 直接做"个体决策"，等于把一个 SE 接近点估计大小的数字当作精确判断。森林估出来的 5735 个 CATE 互相之间的差异，很多落在彼此置信区间的重叠范围内，统计上区分不出谁的效应明显大于谁。

距离的第二段来自协变量的不完整性。因果森林学的是"在 RHC 数据已记录的 29 个协变量层内"的处理效应。一位真实的 ICU 患者床旁决策中医生会考虑的远不止这 29 个，包括患者意识水平的实时变化、家属对侵入性操作的态度、当晚 ICU 床位与人手紧张度、可替代的监测手段（无创超声心动图）。这些因素都没有进入 CATE 估计，但临床决策必须把它们权衡进去。

距离的第三段来自迁移性。森林在 1989–1994 年 5 家美国教学医院的 5735 名患者上学到了 CATE 模式，把它套到 2025 年中国 ICU 患者上需要假设：处理-结果关系的协变量分布、医疗实践、抗生素与镇静药物的使用都没有改变这个 CATE 的形状。这个假设在没有现代数据的情况下不能验证，临床决策必须保留怀疑。

> [!IMPORTANT]
> **命题 9.2 — CATE 的可用性层次**
>
> CATE 估计的临床应用按可信度从高到低分四层：
>
> **第一层（最可信）**：分布层面的发现，"CATE 在 96% 患者上为正"，回答"处理是否对几乎所有人都有同方向效应"。
>
> **第二层**：粗子群对比，"APACHE 高的患者 CATE 略高于 APACHE 低的"，回答"哪些大类患者效应更强"。
>
> **第三层**：变量重要性诊断，"胆红素是最强异质性源"，回答"哪些维度上效应差异大"，但不告诉具体方向。
>
> **第四层（最不可信）**：单个体 $\hat\tau(L_i)$ 用于床旁决策，受 SE 大、未测协变量、迁移性三重限制，临床医生不应把这个数字直接读为"该不该插 RHC"。

下面的代码用 ggplot2 把 5735 名患者的 CATE 分布画成直方图，叠加上 ATE 与 CATE 中位数，让读者直观看到点估计 0.044 在分布里的位置。

```r
library(ggplot2)
ggplot(data.frame(tau = tau_hat), aes(x = tau)) +
  geom_histogram(bins = 60, fill = "#4292C6", alpha = 0.7) +
  geom_vline(xintercept = 0, linetype = "dashed", color = "grey40") +
  geom_vline(xintercept = mean(tau_hat),
             linetype = "solid", color = "#EF6548", linewidth = 0.8) +
  annotate("text", x = mean(tau_hat) + 0.005, y = 200,
           label = sprintf("ATE = %.4f", mean(tau_hat)),
           color = "#EF6548", hjust = 0) +
  scale_x_continuous(breaks = seq(-0.05, 0.15, 0.025)) +
  labs(x = "CATE (个体化处理效应估计)", y = "患者数",
       title = "RHC 数据：5735 名 ICU 患者的 CATE 分布") +
  theme_minimal(base_family = "Songti SC")
```

> [!TIP]
> **解读 — CATE 直方图的形状**
>
> 5735 个 CATE 形成一个略偏右、单峰、长尾在正方向的分布。峰值在 0.04 附近，与 ATE 0.044 几乎重合；右尾延伸到 0.13，左尾只到 -0.04 然后截止。这个形状告诉研究者：RHC 在 ICU 患者群体上是"普遍温和有害 + 少数严重有害"的模式，而不是"一半受益一半受害"。临床政策含义是 RHC 不应作为常规监测使用，但识别"严重有害"那 10% 患者比识别"获益"的 3% 患者更具优先级——前者的效应幅度大且统计可识别，后者效应幅度小且置信区间跨过零。

## 9.7 因果森林的几条边界

> [!WARNING]
> **雷区 — CATE 的推断比 ATE 难得多**
>
> ATE 的 95% CI 由影响函数的样本均值方差给出，标准误典型在 0.012–0.014 这个量级。CATE 的 95% CI 是 individual-level 的，每个 $\hat\tau(L_i)$ 的 SE 通常在 0.04–0.10 这个量级，远大于 CATE 自身的标准差 0.024。换句话说，森林给出的 5735 个 CATE 之间的差距很多时候在统计噪声范围内，单个个体的 $\hat\tau(L)$ 不能直接用于临床决策。CATE 主要的价值在分布层面——整体形状是否偏离同质处理效应——而非给某位具体患者下定论。

> [!WARNING]
> **雷区 — 亚组分析的多重比较与挑樱桃**
>
> 把 5735 个 CATE 按各种维度切，总能找到一些子群有"显著的"获益或受害。这些后验找出来的亚组结论受多重比较污染，不是预先注册的假设。Best Linear Projection 是一种相对稳健的工具，因为它把异质性投影到几个预先指定的协变量线性组合上，比"在 100 个亚组里随便挑几个出来报告"更可信。

> [!WARNING]
> **雷区 — 个体化 CATE ≠ 个体化决策**
>
> 一位 78 岁 APACHE 62 的患者，因果森林给出 $\hat\tau = +0.085$ 这种数字。把它直接当作"不要给这位患者插 RHC"的依据是危险的：第一， $\hat\tau$ 自身有不小的标准误；第二，这位患者的具体临床情境（家属态度、诊断不确定性、监测替代方案）远比模型考虑的协变量丰富；第三，模型在训练集上学到的关系不一定外推到未来类似患者身上（迁移性问题）。CATE 是研究层面的发现工具，不是床旁决策工具。

## 本章知识地图

<div align="center">

| 核心概念 | 核心内容 | 常见误解 | 为什么错 |
|:--|:--|:--|:--|
| ATE vs CATE | ATE 是 CATE 的全人群均值，CATE 是个体效应 | ATE 足够代表所有人 | CATE 方向相反或幅度差异大时 ATE 误导政策 |
| 诚实分裂 | 训练数据分为决定树结构与估计叶子效应两份不重叠的子样本 | 把全部数据用于结构与估计更高效 | 同一份数据双重利用导致叶子效应过拟合，CI 公式失效 |
| 双重稳健评分 | 用 AIPW 影响函数 $\Gamma_i$ 替代 $Y_i$ 作为分裂目标 | 直接用 Y 拟合森林就够 | 普通随机森林优化预测精度，不优化处理效应异质性，找不到真正的 CATE 边界 |
| 变量重要性 | 各协变量被森林分裂规则使用的频率 | 重要性高 = 因果机制 | 重要性是描述性指标，可能反映多个隐藏机制的代理作用 |
| Best Linear Projection | 用 CATE 对预先指定的协变量线性回归，给出主异质性方向 | BLP 是 CATE 的因果分解 | BLP 是描述性投影，不区分因果效应修饰与协变量分布差异 |
| 个体 CATE 标准误 | 单个 $\hat\tau(L_i)$ 的 SE 远大于 CATE 自身标准差 | CATE 数字可以直接做临床决策 | 噪声往往掩盖真实差异，CATE 主要价值在分布层面而非个体层面 |

</div>
